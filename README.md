# 智能值班表排班系统 (Duty Scheduler)

这是一个专为部门设计的智能排班助手，可以读取 PDF 课表并自动筛选出空闲人员，解决跨校区通勤、职位限制等排班难题。

## ✨ 主要功能
- **批量课表解析**：支持一次性上传几十份 PDF 课表（不同专业、不同年级均可），系统会自动识别。
- **智能空闲筛选**：
    - **通勤规则**：自动检测南校区同学，如果下午 16:00 有课，则不允许安排 14:15 的班次。
    - **职位规则**：下午 16:00 的晚班只允许干部/部长/副部/主任团值班。
- **交互式排班**：在网页上直接通过下拉菜单选择人员，系统只显示有空的人，避免排错。
- **Excel 导出**：一键生成最终的值班表。
- **值班记录**：自动统计每个人的值班次数。

## 📦 给普通用户使用

已经打好的 Windows 发布包在 `dist/DutyScheduler-Windows-20260517.zip`。

使用方式：
1. 把 zip 发给对方。
2. 对方解压后，双击 `DutyScheduler.exe`。
3. 不要只单独拷贝 exe；旁边的 `_internal` 文件夹必须保留。
4. 启动后会打开浏览器页面，黑色命令窗口是程序服务窗口，使用时不要关闭。

## 💾 数据保存与清空

软件会把导入后的人员名单、已解析课表和排班记录自动保存在软件文件夹中：

```text
DutyScheduler\data\app_state.json
```

也就是 zip 解压到哪里，数据就存在哪里。如果把整个 `DutyScheduler` 文件夹移动到别的位置，这份数据也会跟着移动。下一次打开软件会自动加载这份数据，不需要重复导入。需要重新导入时，可以在“数据导入”页面底部的“当前数据管理”里清空人员名单、课表、排班记录或全部数据；也可以删除某一份已导入的课表。

## 🛠️ 开发环境运行

### 第一步：安装环境
在当前文件夹下打开终端（Terminal），运行以下命令安装必要的库：
```bash
pip install -r requirements.txt
```

### 第二步：准备数据
1. **人员名单 (Excel)**：
   - 创建一个 Excel 文件（或者使用软件内提供的模板）。
   - 推荐包含以下列名：`Name` (姓名), `Class` (专业班级), `Department` (部门), `Role` (职务/职位), `Campus` (校区)。
   - 也支持中文表头：姓名、专业班级/班级、部门、职务/职位、校区。
2. **课表文件 (PDF)**：
   - 收集所有部员的课表 PDF。
   - **重要**：文件名必须包含学生姓名，例如 `田宇(2023-2024-2)课表.pdf`。

### 第三步：运行软件
在终端输入以下命令启动：
```bash
streamlit run duty_app.py
```
运行后，浏览器会自动打开操作界面。

## 📖 使用流程
1. **数据导入**：进入“数据导入”页面，先上传人员名单，然后**批量选择**所有 PDF 课表并上传。
   - 课表支持三种导入方式：多选 PDF、选择课表文件夹、填写本机文件夹路径。
2. **排班制作**：进入“排班制作”页面，选择你要排的周次（如第5周）。
3. **选择人员**：在表格中点击下拉框，选择合适的人员（系统已自动过滤冲突人员）。
4. **保存与导出**：点击“保存当前周排班”，最后点击“下载值班表”即可。

## 💡 常见问题
- **Q: 不同专业的课表格式不一样怎么办？**
  - A: 只要是学校教务系统导出的标准 PDF 课表，系统都能识别。
- **Q: 为什么某个人在下拉框里找不到？**
  - A: 请检查“查询谁有空”页面，可能是因为他那个时间有课，或者触发了通勤/职位限制规则。
